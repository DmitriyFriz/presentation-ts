Features of TypeScript

  • TypeScript is just JavaScript
  • TypeScript supports other JS libraries
  • JavaScript is TypeScript
  • TypeScript is portable

TypeScript and ECMAScript

Boolean

              
                let isDone: boolean = false;

Number

              
                let decimal: number = 6;
                let hex: number = 0xf00d;
                let binary: number = 0b1010;

String

              
                const color: string = 'blue';

Array

              
                let list: number[] = [1, 2, 3];
                let list: Array = [1, 2, 3];

Tuple

              
                let x: [string, number];
                // Initialize it
                x = ["hello", 10]; // OK

Any

              
                let notSure: any = 4;
                notSure = "maybe a string instead";
                notSure = false; // okay, definitely a boolean

Enum

              
                enum Color {Red, Green, Blue}
                let c: Color = Color.Green;

Void

              
                function warnUser(): void {
                  console.log("This is my warning message");
                }

General TypeScript Advantages

  • TypeScript is an open-source language
  • TypeScript runs on any browser or JavaScript engine
  • TypeScript is similar to JavaScript and uses the same syntax and semantics
  • TypeScript is also closer in syntax to back-end languages like Java and Scala
  • TypeScript has support for the latest JavaScript features from ECMAScript 2015

TypeScript is JavaScript that scales!
